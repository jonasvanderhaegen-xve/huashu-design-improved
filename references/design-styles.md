## I. Information Architecture School (01-04)
> Philosophy: "Data is not decoration — it is building material"

### 01. Pentagram — Michael Bierut Style
**Philosophy:** Typography is language, the grid is thought.
**Key characteristics:**
- Extreme color restraint (black and white + 1 brand color)
- Modern interpretation of the Swiss grid system
- Typographic composition as primary visual language
- Strategic use of negative space (60%+ whitespace)

**Prompt DNA:**
```
Pentagram/Michael Bierut style:
- Extreme typographic hierarchy, Helvetica/Univers family
- Swiss grid with precise mathematical spacing
- Black/white + one accent color (#HEX)
- Information architecture as visual structure
- 60%+ whitespace ratio
- Data visualization as primary decoration
```

**Notable works:** Hillary Clinton 2016 campaign identity
**Search terms:** pentagram hillary logo system

**Key characteristics:**

**Prompt DNA:**
```
Josef Müller-Brockmann Swiss modernism:
- Mathematical grid system (8pt baseline)
- Strict alignment (flush left or centered)
- Two-color maximum (black + one accent)
- Akzidenz-Grotesk or similar rationalist typeface
- No decorative elements
- Timeless, objective aesthetic
```

## Prompt Usage Guide

**Combination formula:** `[style prompt DNA] + [scene template (see scene-templates.md)] + [specific content]`

### Core Principle: Describe Mood, Not Layout

The key to AI image generation: short prompts beat long prompts. Three sentences of mood and content outperform 30 lines of layout detail.

| Kills diversity | Sparks creativity |
|----------------|------------------|
| Specifying color ratios (60%/25%/15%) | Describing mood ("warm like Sunday morning") |
| Dictating layout positions ("title centered, image on right") | Referencing specific aesthetics ("Pentagram editorial feel") |
| Constraining character poses and expressions | Letting AI interpret the style naturally |
| Listing all visual elements to include | Describing what the audience should feel |

### Good / Bad Examples

**Bad — Over-constrained (AI output is flat and empty):**
```
Professional presentation slide. Dark background, light text.
Title centered at top. Two columns below. Left column: bullet points.
Right column: bar chart. Colors: navy 60%, white 30%, gold 10%.
Font size: title 36pt, body 18pt. Margins: 40px all sides.
```

**Good — Mood-driven (generates variety with texture):**
```
A data visualization that feels like a Bloomberg Businessweek
editorial spread. The key number "28.5%" should dominate the
composition like a headline. Warm cream tones with sharp black
typography. The data tells a story of dramatic channel shift.
```

### Execution Path Selection

Choose based on the "Best Path" column in the quick-reference table:
- **AI-generated**: Styles with distinct visual elements (06/07/12/13/14/15/16/20) — use Gemini/Midjourney for direct output
- **HTML rendering**: Styles relying on precise typography (01/03/04/10/11/17/18) — code controls data and layout
- **Mixed**: HTML for skeleton layout + AI-generated images/backgrounds (02/05/08/09/19)

### Quality Control

1. ❌ Don't write "in the style of Pentagram" directly → ✅ Use specific design characteristics instead
2. Text in AI generation often misfires → replace text after generation
3. Proportions distort easily → specify aspect ratio explicitly
4. Generate 3–5 variants first, then refine the best one

**Default aesthetic no-go zones** (override with your own brand as needed):
- ❌ Cyberpunk neon / dark blue backgrounds (#0D1117)
- ❌ Personal attribution or watermarks on cover images
