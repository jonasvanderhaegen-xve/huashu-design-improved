## Usage Guidelines

### Volume Recommendations (from apple-gallery-showcase.md dual-track audio)
- **SFX main track:** `1.0` (no attenuation)
- **BGM background track:** `0.4–0.5` (SFX clearly punches through)
- **Multiple SFX layered:** use `amix=inputs=N:duration=longest:normalize=0` to preserve dynamic range

### ffmpeg Mix Template
```bash
# single SFX aligned to a timestamp:
ffmpeg -i video.mp4 -itsoffset 2.5 -i sfx/ui/click.mp3 \
  -filter_complex "[0:a][1:a]amix=inputs=2:duration=longest:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4

# multiple SFX + BGM:
ffmpeg -i video.mp4 \
  -itsoffset 1.0 -i sfx/transition/whoosh-fast.mp3 \
  -itsoffset 1.6 -i sfx/impact/logo-reveal-v2.mp3 \
  -i bgm.mp3 \
  -filter_complex "[3:a]volume=0.4[bgm];[0:a][1:a][2:a][bgm]amix=inputs=4:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4
```

### Selection Decision Tree
1. **Tactile actions** (typing / clicking / swiping) → `keyboard/` or `ui/`
2. **Element entrance / exit** → `transition/`
3. **Container-level operations** (cards / modals) → `container/`
4. **State feedback** (success / failure / notification) → `feedback/`
5. **Progress / time passing** → `progress/`
6. **Brand impact / key moment** → `impact/`
7. **AI magic / transform** → `magic/`
8. **Command-line / code demo** → `terminal/`

### Avoid Sound Stacking
- Maximum `2 SFX` concurrent at any single time point
- Can run 3 when BGM is below 0.3
- Clear all other SFX for brand impact moments (leave 0.2s silence before the hit)

## Prompt Writing Principles (for reuse)

Reference style: `apple keynote, tight, minimal, no reverb unless ambient, crisp, elegant`

**Three elements of a good prompt:**
1. **Physical sound description:** what object, what action ("mechanical keyboard single key press")
2. **Texture / style qualifier:** apple-style / ios-style / cinematic / retro
3. **Anti-example exclusions:** no reverb / clean studio / minimal

❌ "click sound"
✅ "crisp ui button click, clean modern interface sound, apple-style, high pitched"

❌ "magic"
✅ "bright twinkling stars sound, high pitched glittery chime, fairy dust"
