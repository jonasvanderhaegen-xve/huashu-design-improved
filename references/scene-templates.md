## 1. WeChat Article Cover / Header Image

**Specifications:**
- Cover image: 2.35:1 (900×383px or 1200×510px)
- Body illustration: 16:9 (1200×675px) or 4:3 (1200×900px)

**Key design elements:**
- Visual impact first (users scan quickly in a feed)
- Minimal or no text (the WeChat title will overlay it)
- Moderate color saturation (WeChat reading environment is predominantly white)
- Avoid excessive detail (must be recognizable as a thumbnail)

**Recommended styles:** 01 Pentagram / 11 Build / 12 Sagmeister / 18 Kenya Hara / 07 Field.io

**Scene prompt template:**
```
[style DNA here]
- Article cover image for WeChat subscription
- Landscape format, 2.35:1 aspect ratio
- Bold visual impact, minimal or no text
- Moderate color saturation for white reading environment
- Must remain recognizable as thumbnail
- Clean composition with clear focal point
```

## 4. PPT / Keynote Presentation

**Specifications:**
- Standard: 16:9 (1920×1080px)
- Widescreen: 16:10 (1920×1200px)

**Key design elements:**
- One core message per slide (no cramming)
- Clear type hierarchy (title 40pt+ / body 24pt+ / caption 16pt+)
- Generous whitespace — clearer when projected
- Visual-to-text ratio at least 60:40
- Consistent visual system (color, typeface, spacing)

**Recommended styles:** 01 Pentagram / 10 Müller-Brockmann / 11 Build / 18 Kenya Hara / 04 Fathom

**Scene prompt template:**
```

- Presentation slide design, 16:9
- One core message per slide
```
